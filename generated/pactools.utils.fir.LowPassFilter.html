pactools.utils.fir.LowPassFilter¶

class pactools.utils.fir.LowPassFilter(fs, fc, bandwidth, ripple_db=60.0)[source]¶

Low-pass FIR filter

Designs a FIR filter that is a low-pass filter.

Parameters:

fs : float

Sampling frequency

fc : float

Cut-off frequency of the low-pass filter

bandwidth : float

Bandwidth of the FIR wavelet filter

ripple_db : float (default 60.0)

Positive number specifying maximum ripple in passband (dB) and minimum ripple in stopband, in Kaiser-window low-pass FIR filter.

Examples

>>> from pactools.utils import LowPassFilter
>>> f = LowPassFilter(fs=100., fc=5., bandwidth=1.)
>>> f.plot()
>>> signal_out = f.transform(signal_in)
